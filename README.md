This is a **Java-based backend** for an Airbnb-inspired application, built using **Spring Boot 3**. It provides a robust set of **RESTful APIs** to manage bookings, reservations, user authentication, and house search functionality. 

Key highlights:
- **OAuth2 authentication with Auth0** for secure **role-based access control (RBAC)**.
- Designed using **Domain-Driven Design (DDD)** for scalability and maintainability.

---

## ‚ú® Features

- **User Management:** Secure Auth0-based login and RBAC for tenants and landlords.
- **Booking Management:** Create, view, and cancel bookings and reservations.
- **House Search:** Filter houses by location, date, guests, and number of beds.
- **Role-Based Access Control:** 
  - **Tenants:** Book and search houses.  
  - **Landlords:** Manage property listings.
- **RESTful API Design:** Clean and efficient API architecture.
- **Logging:** SLF4J integration for application monitoring.
- **Scalable Architecture:** Follows Domain-Driven Design principles.
